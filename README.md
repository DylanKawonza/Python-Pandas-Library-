# Python-Pandas-Library-
🐍 Python Pandas Data Analysis Project
Reflective Summary – Data Technician Bootcamp
This repository contains my Python data analysis work completed during the Level 3 Data Technician Bootcamp, where I learned how to use pandas, NumPy, and matplotlib to explore, clean, and analyse retail and sales datasets.
This project helped me understand how Python can be used as a powerful tool for data manipulation and insight generation.

🧠 What I Learned
📦 Working With Pandas DataFrames
I learned how to load and explore datasets using:

pd.read_csv() / pd.read_excel()

.head() / .tail()

.info() / .describe()

These tools helped me quickly understand the structure and quality of the data.

🔍 Filtering & Subsetting Data
I practised selecting specific rows and columns using:

.loc[] — label‑based selection

.iloc[] — position‑based selection

Boolean filtering (e.g., df[df["Price"] > 50])

This allowed me to isolate key information such as high‑value products, specific customer segments, and filtered sales records.

🛠️ Data Cleaning & Manipulation
I used a range of pandas methods to prepare and clean data:

.dropna() — removing missing values

.fillna() — replacing missing values

.sort_values() — sorting by columns

.groupby() — summarising data

.value_counts() — frequency analysis

.rename() — cleaning column names

.astype() — converting data types

These skills helped me transform messy datasets into clean, usable tables ready for analysis.

📊 Data Analysis & Aggregation
Using pandas, I explored retail and sales data by:

Grouping sales by product, category, or region

Calculating totals, averages, and counts

Identifying trends and patterns

Comparing performance across different segments

This taught me how to extract meaningful insights from raw data.

📈 Data Visualisation
I created visualisations using:

Pandas built‑in .plot()

matplotlib.pyplot

Including:

Bar charts

Line charts

Histograms

Scatter plots

These visuals helped me communicate findings clearly and spot trends more easily.
